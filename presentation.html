The SLA & Operational Challenge

Problem Space
  • Inline Card Transaction Authorization: Decisions must execute within the network switch context before responding to payment switches.
  • Incomplete Information: Long-tail outcomes (chargebacks, customer disputes) arrive days later. Decisions must run on local history.
  • Asymmetric Error Costs:
    • False Positive: Blocks legitimate user transaction (lost revenue & customer friction).
    • False Negative: Approves fraud transaction (chargeback costs & regulatory exposure).

The MCP Gateway Layer

Infrastructure
  • Tool Abstraction: Isolates agents from directly referencing API endpoints. All lookups conform to standardized Model Context Protocol JSON-RPC.
  • Authentication & Authorization: Enforces strict tenant filters and domain keys to prevent unauthorized tool access.
  • Tracing & Budget Tracking: Active transaction IDs propagate to downstream databases to profile performance.

SLA Circuit Breakers

Gateway wraps calls in short-circuit triggers. If a resource server (e.g. Geo IP service) times out 3 times in a row, the gateway temporarily bypasses it for 30s, keeping the orchestration loop responsive.

Constitutional Governance

Rules

Policy Rules

  • no_policy_override: true
  • no_hallucinated_facts: true
  • evidence_required: true
  • max_agent_hops: 5
  • max_tool_calls: 3
  • budget_enforced: true
  • deterministic_output: true
  • no_self_deployment: true

Policy Always Wins

Even with sophisticated agent calculations, the final outcome must respect hard constitutional rules. Ambiguity is resolved safely by defaulting to **ESCALATE** (OTP or human review) rather than approving unverified patterns.

Asynchronous Feedback & Replay Loops

Optimization

Kafka Event Streaming

Decisions, audit logs, chargebacks, and manual verdicts are streamed to Kafka topics.

Feedback Agent Cluster

Correlation Agent: Resolves chargebacks to past transactions.
Pattern Mining Agent: Scans clusters for emerging risk shapes.
Threshold Optimizer: Recommends parameter modifications.

Controlled Rollout Pipeline

Threshold changes cannot auto-deploy. They pass:
1. Governance Agent Review
2. Shadow Mode (in parallel to production)
3. Progressive Canaries (1% → 5% → 25% → 50%)
4. Blue-Green Production Promotion
